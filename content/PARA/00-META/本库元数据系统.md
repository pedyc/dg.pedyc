## 笔记默认元数据

参见 [[template_meta]]

| 字段          | 含义              | 约束 | 备注                                                                                     |
| ------------- | ----------------- | ---- | ---------------------------------------------------------------------------------------- |
| title         | 笔记标题          | 必须 | 文件名，唯一<br>命名规范参见 [[本库命名规范]]                                            |
| aliases       | 笔记别名          |      | 展示标题                                                                                 |
| uid           | UID               | 必须 | 唯一标识                                                                                 |
| date-created  | 创建日期          | 必须 |                                                                                          |
| date-modified | 修改日期          | 必须 |                                                                                          |
| tags          | 标签列表          |      |                                                                                          |
| para          | PARA 分类         | 必须 | { project \| area \| resource \| archive }                                               |
| zettel        | Zettelkasten 分类 | 必须 | { flash \| literature \| permanent \| moc }                                              |
| content-type  | 笔记内容分类      | 必须 | { term \| concept \| method \| theory \| record }<br>详情参见 [[本库笔记分类与内容模板]] |
