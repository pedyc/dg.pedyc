## PARA 全局索引

```dataview
TABLE WITHOUT ID
  file.link AS "区域",
  choice(file.path = "10-PROJECTS", "🛠️", 
         file.path = "20-AREAS", "🧠", 
         "📦") AS "图标",
  length(rows) AS "关联原子笔记数"
FROM "_atomic-notes"
WHERE contains(file.folder, this.file.folder)
GROUP BY file.folder
```
