## 🔎描述

JavaScript 是一种高级的、解释型的编程语言，主要用于客户端 Web 开发，为网页添加交互性和动态效果。它也可以用于服务器端开发（Node.js）、移动应用开发（React Native、Ionic）和桌面应用开发（Electron）。
