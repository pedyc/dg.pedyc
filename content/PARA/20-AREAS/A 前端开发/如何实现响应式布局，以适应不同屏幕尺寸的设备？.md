## 回答

实现响应式布局，以适应不同屏幕尺寸的设备，可以采用以下方法：

1. **Meta 标签**：
		* 在 HTML 文档的 `<head>` 标签中添加 viewport meta 标签，以控制页面的缩放和显示比例。
		* 常用的 viewport meta 标签设置如下：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

2. **CSS Media Queries**：
		* 使用 CSS Media Queries 根据不同的屏幕尺寸应用不同的 CSS 样式。
		* Media Queries 可以根据屏幕宽度、高度、设备像素比等条件来应用不同的样式。

3. **弹性布局（Flexbox）和网格布局（Grid）**：
		* 使用 Flexbox 和 Grid 创建灵活的页面布局，以适应不同的屏幕尺寸。
		* Flexbox 适用于一维布局，而 Grid 适用于二维布局。

4. **流式布局**：
		* 使用百分比或 auto 等单位，使元素的大小根据屏幕尺寸自动调整。
		* 避免使用固定宽度，以确保元素在不同屏幕尺寸下都能正常显示。

5. **响应式图片**：
		* 使用 `<picture>` 元素或 `srcset` 属性，根据不同的屏幕尺寸加载不同大小的图片。
		* 可以使用 `sizes` 属性来指定不同屏幕尺寸下图片的显示大小。

## 示例

1. **使用 CSS Media Queries**：

```css
/* 默认样式 */  
body {  
	font-size: 16px;  
}

/* 屏幕宽度小于 768px 时的样式 */  
@media (max-width: 768px) {  
	body {  
		font-size: 14px;  
	}  
}

/* 屏幕宽度大于 768px 时的样式 */  
@media (min-width: 768px) {  
	body {  
		font-size: 18px;  
	}  
}
```

2. **使用 Flexbox**：

```css
.container {
	display: flex;
	flex-wrap: wrap; /* 允许元素换行 */
}

.item {
	width: 50%; /* 在小屏幕上占据 50% 的宽度 */
}

@media (min-width: 768px) {
.item {
		width: 25%; /* 在大屏幕上占据 25% 的宽度 */
	}
}

```
