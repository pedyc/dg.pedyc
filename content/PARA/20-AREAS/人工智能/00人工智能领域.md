## [[00人工智能领域]]

### 🧱 核心组件 (Core Components)

* **[[LLM概述]] (领域基石):**
	* **概述:** 大型语言模型 (LLM) 是指具有数百万或数十亿参数的深度学习模型，能够理解和生成人类语言。
	* **核心概念:**
		* Transformer: LLM 的基础架构。
		* 预训练: 在大量文本数据上训练模型。
		* 微调: 在特定任务上进一步训练模型。
	* **相关技术:**
		* 自然语言处理 (NLP)
		* 深度学习 (DL)
	* **发展趋势:**
		* 多模态 LLM
		* 可解释性 LLM
* **[[MCP协议]] (高频工具):**
	* **概述:** MCP 协议 (Meta-Cognitive Prompting) 是一种利用元认知能力来改进 LLM 性能的提示方法。
	* **核心概念:**
		* 元认知: 对认知的认知，包括对自身认知过程的理解和控制。
		* 提示词: 引导 LLM 生成期望输出的文本。
		* 反思: 让 LLM 对自身的输出进行评估和改进。
	* **应用场景:**
		* 提高 LLM 的准确性和可靠性。
		* 解决 LLM 的幻觉问题。
		* 增强 LLM 的创造力。
	* **局限性:**
		* 计算成本高
		* 需要大量数据
* **[[提示词工程]] (高频工具):**
	* **概述:** 提示词工程是一种设计和优化提示词的技术，用于引导 LLM 生成期望的输出。
	* **核心概念:**
		* 提示词: 引导 LLM 生成期望输出的文本。
		* 提示词模板: 可复用的提示词结构。
		* 提示词策略: 设计提示词的方法和技巧。
	* **常用技巧:**
		* 清晰明确的指令
		* 提供上下文信息
		* 使用示例
		* 限制输出格式
	* **评估指标:**
		* 准确率
		* 召回率
		* F1 值
* **[[AI 在前端的应用]] (最佳实践):**
	* **用户体验优化:**
		* 智能表单: 使用 AI 技术自动填充表单，提高用户效率。
		* 智能搜索: 使用 AI 技术提升搜索结果的准确性和相关性。
		* 智能推荐: 使用 AI 技术为用户推荐个性化的内容。
		* 智能客服: 使用 AI 技术构建智能客服机器人，提供 24 小时在线服务。
	* **开发效率提升:**
		* 代码自动生成: 使用 AI 技术自动生成代码，提高开发效率。
		* UI 自动化测试: 使用 AI 技术自动进行 UI 测试，提高测试效率。
* **[[AI 辅助前端工程化]] (最佳实践):**
	* **AI 辅助代码审查:** 利用 AI 模型进行代码风格检查和潜在 Bug 识别 (例如：使用 CodeQL)。
	* **AI 驱动的性能优化:** 使用 AI 模型分析前端性能数据，并提供优化建议 (例如：使用 PageSpeed Insights API)。
	* **AI 辅助自动化测试:** 利用 AI 模型自动生成测试用例，提高测试覆盖率 (例如：使用 Diffy)。
