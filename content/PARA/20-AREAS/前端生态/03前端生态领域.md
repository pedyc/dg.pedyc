## 前端生态 Roadmap

### 概述

本学习路线旨在帮助开发者系统地学习前端框架及其生态系统，构建完整的前端知识体系，并为技术选型和职业发展提供参考。

### 学习目标

* 掌握主流前端框架的核心概念和使用方法。
* 了解前端生态系统的组成和常用工具。
* 能够根据项目需求选择合适的技术栈。
* 具备独立开发和维护前端应用的能力。

### 学习内容

1. **基础知识**
	* **HTML**
			* 学习目标：掌握 HTML 语义化标签、常用元素和属性。
			* 学习资源：[[HTML 官方文档]]、[[MDN Web Docs]]
	* **CSS**
			* 学习目标：掌握 CSS 常用属性、选择器、盒模型和布局方式。
			* 学习资源：[[CSS 官方文档]]、[[MDN Web Docs]]
	* **JavaScript**
			* 学习目标：掌握 JavaScript 基础语法、DOM 操作、ES6+ 新特性。
			* 学习资源：[[JavaScript 官方文档]]、[[MDN Web Docs]]
	* **前端构建工具（Webpack、Parcel、Rollup）**
			* 学习目标：了解前端构建工具的作用和原理，掌握 Webpack 的基本配置和使用。
			* 学习资源：[[Webpack 官方文档]]、[[Parcel 官方文档]]、[[Rollup 官方文档]]
	* **版本控制（Git）**
			* 学习目标：掌握 Git 的基本命令和工作流程，能够使用 Git 进行版本控制和团队协作。
			* 学习资源：[[Git 官方文档]]、[[Pro Git Book]]
2. **主流框架**
	* **React**
			* 学习目标：掌握 React 的核心概念（组件、JSX、状态管理、生命周期），能够使用 React 构建用户界面。
			* 学习资源：[[React 官方文档]]、[[React Tutorial]]
	* **Vue**
			* 学习目标：掌握 Vue 的核心概念（模板语法、组件、指令、计算属性、侦听器），能够使用 Vue 构建用户界面。
			* 学习资源：[[Vue 官方文档]]、[[Vue Mastery]]
	* **Angular**
			* 学习目标：掌握 Angular 的核心概念（模块、组件、服务、依赖注入、路由），能够使用 Angular 构建大型单页应用。
			* 学习资源：[[Angular 官方文档]]、[[Angular Tutorial]]
3. **状态管理**
	* **Redux**
			* 学习目标：掌握 Redux 的核心概念（Store、Action、Reducer），能够使用 Redux 管理应用状态。
			* 学习资源：[[Redux 官方文档]]、[[Redux Toolkit]]
	* **Vuex**
			* 学习目标：掌握 Vuex 的核心概念（State、Mutation、Action、Getter），能够使用 Vuex 管理 Vue 应用状态。
			* 学习资源：[[Vuex 官方文档]]
	* **NgRx**
			* 学习目标：掌握 NgRx 的核心概念（Action、Reducer、Effect、Selector），能够使用 NgRx 管理 Angular 应用状态。
			* 学习资源：[[NgRx 官方文档]]
4. **路由管理**
	* **React Router**
			* 学习目标：掌握 React Router 的基本使用方法，能够使用 React Router 实现页面路由和导航。
			* 学习资源：[[React Router 官方文档]]
	* **Vue Router**
			* 学习目标：掌握 Vue Router 的基本使用方法，能够使用 Vue Router 实现页面路由和导航。
			* 学习资源：[[Vue Router 官方文档]]
	* **Angular Router**
			* 学习目标：掌握 Angular Router 的基本使用方法，能够使用 Angular Router 实现页面路由和导航。
			* 学习资源：[[Angular Router 官方文档]]
5. **UI 组件库**
	* **Ant Design**
			* 学习目标：掌握 Ant Design 的基本使用方法，能够使用 Ant Design 构建美观易用的用户界面。
			* 学习资源：[[Ant Design 官方文档]]
	* **Material UI**
			* 学习目标：掌握 Material UI 的基本使用方法，能够使用 Material UI 构建符合 Material Design 规范的用户界面。
			* 学习资源：[[Material UI 官方文档]]
	* **Element UI**
			* 学习目标：掌握 Element UI 的基本使用方法，能够使用 Element UI 构建美观易用的用户界面。
			* 学习资源：[[Element UI 官方文档]]
6. **测试**
	* **Jest**
			* 学习目标：掌握 Jest 的基本使用方法，能够使用 Jest 进行单元测试和集成测试。
			* 学习资源：[[Jest 官方文档]]
	* **Mocha**
			* 学习目标：掌握 Mocha 的基本使用方法，能够使用 Mocha 进行单元测试和集成测试。
			* 学习资源：[[Mocha 官方文档]]
	* **Cypress**
			* 学习目标：掌握 Cypress 的基本使用方法，能够使用 Cypress 进行端到端测试。
			* 学习资源：[[Cypress 官方文档]]
7. **部署**
	* **Netlify**
			* 学习目标：掌握 Netlify 的基本使用方法，能够使用 Netlify 部署前端应用。
			* 学习资源：[[Netlify 官方文档]]
	* **Vercel**
			* 学习目标：掌握 Vercel 的基本使用方法，能够使用 Vercel 部署前端应用。
			* 学习资源：[[Vercel 官方文档]]
	* **AWS**
			* 学习目标：掌握 AWS 的基本使用方法，能够使用 AWS 部署前端应用。
			* 学习资源：[[AWS 官方文档]]
