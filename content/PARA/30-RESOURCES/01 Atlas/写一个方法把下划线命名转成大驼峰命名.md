## 题目

https://github.com/haizlin/fe-interview/issues/12

写一个方法把下划线命名转成大驼峰命名

## 实现

思路：线性扫描

> [!hint] 复杂度分析
> - 时间复杂度：$O(n)$
> - 空间复杂度：$O(n)$

```javascript
var toCamel = function (name) {
    return name.split('_').map((word) =>
        word.charAt(0).toUpperCase() + word.slice(1)
    ).join('')
}
```
