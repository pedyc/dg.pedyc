## 定义

LCP 衡量的是在页面首次开始加载时，最大可见元素完成渲染的时间。

## 影响因素

* 服务器响应时间
* 渲染阻塞的 JavaScript 和 CSS
* 资源加载时间
* 客户端渲染

## 优化方法

* 优化服务器响应时间
* 优化 CSS 和 JavaScript
* 优化图片
* 使用 CDN

## 良好 LCP

2.5 秒以内
