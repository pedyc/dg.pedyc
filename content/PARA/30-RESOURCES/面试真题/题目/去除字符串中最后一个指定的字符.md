## 题目

https://github.com/haizlin/fe-interview/issues/9

去除字符串中最后一个指定的字符

## 实现

思路：线性扫描

> [!hint] 复杂度分析
> - 时间复杂度：$O(n)$
> - 空间复杂度：$O(n)$

```javascript
function delLast(str, char) {
    if (typeof str === 'string') {
        let idx = str.lastIndexOf(char)
        str = str.substring(0, idx) + str.substring(idx + 1, str.length)
    }
    return str
}
```
