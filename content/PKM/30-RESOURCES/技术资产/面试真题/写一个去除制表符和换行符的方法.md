写一个去除制表符和执行符的方法

## 实现

思路：正则替换
注意：制表符\t，换行符\n，回车符\r

> [!hint] 复杂度分析
> - 时间复杂度：$O(n)$
> - 空间复杂度：$O(1)$

```javascript
var func = function(str){
	return str.replace(/\t|\n/g,'')
}
```
