写一个方法去掉字符串间的空格

## 实现

思路：正则替换
注意：正则匹配

> [!hint] 复杂度分析
> - 时间复杂度：$O(n^2)$
> - 空间复杂度：$O(1)$

```javascript
const TRIM_POSITION = Object.freeze({
	ALL: /\s*/g,
	LEFT: /^\s*/g,
	RIGHT: /\s*$/g,
	MIDDLE: /(?!^\s+)\s+(?!\s+$)/g,
	BOTH: /^\s*|\s*$/g
})

var trim = function(s,pos = TRIM_POSITION.BOTH){
	s = s.replace(pos,'');
	return s;
}
```
